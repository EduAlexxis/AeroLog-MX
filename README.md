AeroLog is a modern utility application designed for aviation maintenance professionals. It streamlines the process of logging inventory, tracking parts, and documenting maintenance actions.

Built with the latest Apple technologies, AeroLog moves beyond simple data entry by integrating OCR (Optical Character Recognition) to instantly scan part tags and CloudKit for secure, private synchronization across devices.

AeroLog utilizes a Zero-Knowledge Architecture.

No Developer Access: We do not have servers. We cannot see your data.

Private Cloud: All sync happens via your personal iCloud Private Database.

Local First: Supports full offline functionality using SwiftData.

‚ú® Key Features

Intelligent Scanner: Uses the Apple Vision Framework to scan physical part tags and automatically extract Part Numbers (PN) and Serial Numbers (SN).

Visual Evidence: Attach high-resolution photos to your maintenance logs.

Seamless Sync: Built on CloudKit to keep your iPad and iPhone in sync automatically.

Offline Ready: Powered by SwiftData for robust local storage when the hangar has no Wi-Fi.

Ad-Free: A professional tool with no tracking, analytics, or advertisements.
